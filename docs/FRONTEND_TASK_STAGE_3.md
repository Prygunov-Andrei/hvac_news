# Задание для Фронтенд-разработчика (Этап 3: Лента Новостей)

**Цель этапа:** Реализовать отображение новостной ленты и детальной страницы новости.

**API Base URL:** `https://<your-ngrok-url>/api`

## 1. Список новостей (Лента)
**URL:** `/news`
**API:** `GET /api/news/`

**Функционал:**
*   Бесконечная лента или пагинация.
*   Отображение карточек новостей.
*   В карточке:
    *   Заголовок (`title` - автоматически берется на нужном языке).
    *   Дата публикации (`pub_date`).
    *   Краткое содержание (можно обрезать `body` или брать первые N символов).
    *   Первая картинка из `body` (если есть) в качестве обложки.

## 2. Детальная страница новости
**URL:** `/news/[id]`
**API:** `GET /api/news/[id]/`

**Функционал:**
*   Отображение полного текста новости (`body`).
*   **Важно:** Поле `body` содержит **Markdown**. Необходимо использовать библиотеку для рендеринга (например, `react-markdown`).
*   **Обработка медиа:**
    *   Картинки в MD приходят как `![Alt](/media/news/1_img.jpg)`. Рендерер должен их корректно отображать.
    *   Видео приходят как HTML тег `<video controls src="/media/news/1_vid.mp4" ...></video>`. Необходимо разрешить рендеринг HTML внутри Markdown (с осторожностью, или использовать плагин для `rehype-raw`).
    *   Альтернатива для видео: Парсить кастомный тег, если мы решим его использовать. Но сейчас API отдает готовый `<video>` тег.

## 3. Мультиязычность и Fallback
При переключении языка на сайте (Accept-Language header), API будет возвращать `title` и `body` на выбранном языке.

**Обратите внимание:** Если новость не переведена на выбранный язык (например, DE), сервер автоматически вернет контент на **Английском** (EN) или **Русском** (RU) (механизм Fallback).
Фронтенду не нужно об этом беспокоиться, просто отображайте пришедшие поля `title` и `body`.

## Пример ответа API
`GET /api/news/1/`
```json
{
  "id": 1,
  "title": "Заголовок новости",
  "title_ru": "Заголовок новости",
  "title_en": "News Title",
  "body": "Текст новости... <video src='...'></video> ...",
  "pub_date": "2023-12-27T10:00:00Z",
  "media": [
    { "id": 1, "file": "/media/news/1_img.jpg", "media_type": "image" },
    { "id": 2, "file": "/media/news/1_vid.mp4", "media_type": "video" }
  ]
}
```
